 ===================================
|                                   |
|             Controls              |
|                                   |
 ===================================
Left                    left arrow
Right                   right arrow
Up                      up arrow
Down                    down arrow
Change gravity          W
Skip level              T

 ===================================
|                                   |
|             Gameplay              |
|                                   |
 ===================================
Collect the key to advance to the next level, but be sure to avoid the dangerous
inhabitants and obstacles of the world! Gravity is shifted in a clockwise fashion,
and the current direction is always indicated by the excessively large red arrow
in the upper right hand corner.

If you die, you will restart the level. If you run out of lives entirely, you lose.
