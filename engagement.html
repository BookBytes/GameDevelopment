Game Development

Personal Engagement Project 2015

This assignment is worth 5% of your overall grade.

Overview

Because game development is a particularly creative field, we expect every student in the class to participate in ways not explicitly defined by the curriculum and syllabus.

Examples

  • Create a game of your own.
  • Create a piece of media that could be used in a game.
    For instance:
    • 2D or 3D art
    • Original music or sound content
  • Write an essay that challenges a commonly accepted idea or proposes a new one.
  • Build a tool to help game developers
  • Something else!

Deliverables

  1. Proposal
  2. Progress Report #1
  3. Progress Report #2
  4. Final Project

Grading

Grading will be based on effort and quality.
